EXIF AND PHOTO DATA REMOVER
HOW TO USE ?
[INSTALLATION & USAGE]
1. Install Python 3.x and run: pip install Pillow
2. Download the project files.
3. Place 'cleaner.py' in the same folder as the 'Forensic_Eng.exe'.
4. Create an 'EXIF' folder on your Desktop and put your images inside.
5. Run 'Forensic_Eng.exe' and click the button.
6. Check your Desktop for the 'Exif_Removed' folder.
